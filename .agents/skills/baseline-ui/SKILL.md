# Baseline UI

Use for a fast cleanup pass without a full redesign.

## Protocol

1. Normalize spacing between sections
2. Fix alignment and visual hierarchy
3. Remove redundant wrappers and inconsistent padding
4. Ensure consistent button and label styles
