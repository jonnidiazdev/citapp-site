# UI Skills (citapp-site)

Skills instalados desde [ui-skills.com](https://www.ui-skills.com) para guiar diseño y calidad de interfaz.

| Skill | Uso |
|-------|-----|
| `frontend-design` | Dirección visual distintiva, tipografía, evitar estética genérica |
| `interface-design` | Sistemas de diseño, patrones de UI |
| `baseline-ui` | Pasada rápida de spacing, jerarquía y consistencia |
| `fixing-accessibility` | ARIA, teclado, focus, contraste, formularios |

Invocar en chat: `@frontend-design` o pedir rediseño / audit a11y.
