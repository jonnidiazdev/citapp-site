# Interface Design

Use when building or refining dashboard-style UIs with data density, hierarchy, and consistent patterns.

## Protocol

1. Establish clear visual hierarchy (summary → sections → cards)
2. Keep data tables and financial numbers scannable (`font-mono-data`)
3. Use consistent card patterns across asset types
4. Group related actions; avoid scattered CTAs
5. Match spacing rhythm across sections
